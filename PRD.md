## ⚽ 해외파 한국 선수 스탯 아카이브 웹 (PRD)

### 1. 개요 (Introduction)

* **제품명 (가칭):** K-Stats 아카이브 (또는 K-Players Weekly)
* **목적:** 주간 '해외파 선수 활약상' 뉴스레터 독자들에게 요약된 정보(Top 3, 포지션별 Top 1 등)의 원본 데이터를 투명하고 상세하게 제공합니다. 유저는 뉴스레터에서 접한 정보를 확인하고, 더 나아가 자신이 원하는 선수의 세부 스탯을 직접 탐색할 수 있습니다.
* **핵심 가치:** 요약된 인사이트(뉴스레터)와 상세한 원본 데이터(웹사이트)를 연결하여 팬들에게 가장 포괄적인 해외파 선수 스탯 경험을 제공합니다.

### 2. 타겟 유저 (Target Audience)

* **핵심 타겟:** '해외파 선수 활약상' 뉴스레터 구독자. 뉴스레터에서 본 '주간 Top 3'나 '상승세' 선수의 구체적인 평점, 출전 시간, 공격 포인트가 궁금한 사람.
* **확장 타겟:** 특정 해외파 선수의 팬으로, 매주 선수의 상세 스탯을 빠짐없이 확인하고 싶은 축구 팬.

### 3. 제품 목표 (Goals)

* **데이터 접근성:** 유저가 CSV에 담긴 모든 선수의 주간/시즌 데이터를 한눈에 쉽게 찾아볼 수 있도록 합니다.
* **직관적 UI/UX:** 복잡한 기능 없이, 데이터를 '테이블'이라는 가장 친숙하고 직관적인 형태로 제공하여 즉각적인 정보 탐색이 가능하게 합니다.
* **신뢰도 확보:** 뉴스레터에서 제공하는 인사이트의 근거 데이터를 투명하게 공개하여 신뢰도를 높입니다.

### 4. 핵심 기능 (MVP Features)

**F-01: 전체 선수 스탯 테이블 (All Players Stats Table)**

* **설명:** CSV 파일의 모든 선수 데이터를 표시하는 반응형 웹 테이블입니다. 이 테이블은 웹사이트의 메인 페이지 그 자체가 됩니다.
* **데이터 컬럼 (표시 기준):**
    * **핵심 정보:** 선수명 (`player_name_kr`), 소속팀 (`team`), 리그 (`league`), 포지션 (`position`)
    * **주간 활약:** 주간 평점 (`weekly_avg_rating`), 출전 경기 (`weekly_matches`), 출전 시간 (`weekly_minutes`), 골 (`weekly_goals`), 어시스트 (`weekly_assists`)
    * **시즌 누적:** 시즌 평점 (`season_avg_rating`), 출전 경기 (`season_matches`), 골 (`season_goals`), 어시스트 (`season_assists`)
    * **선수 상태:** 부상 여부 (`is_injured`, `injury_status`)
    * **링크:** 상세 정보 (`fotmob_url`) - (새 탭에서 열기)

* **UI/UX 요구사항:**
    * **가독성:** PC와 모바일 환경 모두에서 가로 스크롤을 최소화하거나, 모바일에서는 중요 컬럼(선수명, 팀, 주간 평점, 골/어시)만 우선 노출하는 반응형 디자인을 적용합니다.
    * **데이터 포맷팅:**
        * `weekly_avg_rating` (주간 평점): 소수점 2자리까지 표시. 0점은 '-'로 표시.
        * `is_injured` (부상): 'Yes'일 경우, 선수명 옆에 붉은색 아이콘(예: 🚑)을 표시하고 `injury_status` (예: 'Doubtful')를 툴팁으로 보여줍니다.
    * `fotmob_url`: '상세' 버튼 또는 선수명 자체에 링크를 걸어 유저가 Fotmob 원본 페이지로 바로 이동할 수 있게 합니다.

**F-02: 데이터 정렬 (Sorting)**

* **설명:** 유저가 테이블의 특정 컬럼 헤더(Header)를 클릭하여 데이터를 오름차순/내림차순으로 정렬할 수 있게 합니다.
* **적용 컬럼:** 선수명, 소속팀, 리그, 포지션, 주간 평점, 시즌 평점, 주간/시즌 골, 주간/시즌 어시스트

**F-03: 기본 필터링 (Basic Filtering)**

* **설명:** 테이블 상단에 유저가 원하는 데이터만 골라볼 수 있는 간단한 필터를 제공합니다.
* **필터 옵션 (MVP):**
    1.  **리그 (League):** CSV에 있는 `league` 값(예: Premier League, Bundesliga, Ligue 1 등)을 드롭다운으로 제공합니다. (다중 선택 가능)
    2.  **포지션 (Position):** `position` 값을 기준으로 (예: Attacker, Midfielder, Defender) 필터링합니다. *[참고: CSV의 'position' 컬럼을 '공격수/미드필더/수비수'로 그룹핑하는 데이터 전처리 로직이 필요할 수 있습니다.]*
    3.  **부상 선수만 보기:** '부상' 체크박스

### 5. 데이터 요구사항

* **Source:** `Fotmob-data - Korean_Players_Weekly.csv`
* **업데이트 주기:** 초기 MVP는 정적(static)인 CSV를 기반으로 하지만, 향후 이 데이터가 자동으로 업데이트되는 것을 고려하여 설계합니다. (예: 매주 월요일 CSV 파일 교체)
* **제외 데이터 (MVP):** `recent_matches_json` 컬럼은 복잡한 JSON 구조이므로 초기 테이블 뷰에서는 제외합니다. 이는 향후 '선수 상세 페이지' 기능에서 활용합니다.

### 6. 향후 계획 (Future Scope - Post-MVP)

* **선수 상세 페이지:** 선수를 클릭하면 `recent_matches_json` 데이터를 분석하여 최근 5경기 스탯 변화, 평점 추이 그래프 등을 보여주는 개별 페이지.
* **고급 검색/필터:** 선수 이름으로 직접 검색, '상승세/하락세' (예: 주간 평점 > 시즌 평점) 선수 필터링.
* **뉴스레터 섹션 연동:** '주간 Top 3', '포지션별 Top 1'을 웹사이트 상단에 별도 섹션으로 노출하여 뉴스레터와 동일한 경험을 제공.
